# Smart Planogram

## Overview

Smart Planogram is an AI-powered shelf compliance system that compares a current shelf image against a stored planogram image. The system automatically detects products, identifies them using machine learning, and generates compliance and restocking reports.
[Smart Planogram](https://drive.google.com/drive/folders/1c15aTXThyBCzValCp__v1kyCFnAcd4Fu)

The project combines:

- YOLO Object Detection
- MobileNetV3 Feature Extraction
- FAISS Similarity Search
- Flask Web Application
- Automated Planogram Comparison

---

## Capstone Contribution

This project converts a previously manual notebook-based workflow into a complete web application.

The system automatically:

1. Loads the selected shelf planogram.
2. Detects products from the stored planogram.
3. Detects products from the uploaded shelf image.
4. Identifies products using MobileNetV3 and FAISS.
5. Compares the current shelf against the expected planogram.
6. Generates compliance and restocking reports.

## How to Use

### Step 1: Select Shelf

Choose:

- Shelf A
- Shelf B
- Shelf C
- Shelf D
- Shelf E

The system automatically loads the corresponding stored planogram.

### Step 2: Upload Current Shelf Photo

Upload a recent image of the shelf.

Recommended:

- Good lighting
- Entire shelf visible
- Front-facing image
- Minimal blur

### Step 3: Run Analysis

Click **Run Analysis**.

The system will:

- Detect products
- Identify products
- Compare against planogram
- Generate compliance report
- Generate restocking recommendations

## Current Limitations

### Prototype System

This project is currently a prototype intended for educational and demonstration purposes.

### Image Quality Dependency

Performance may decrease when:

- Images are blurry
- Lighting is poor
- Products are partially hidden
- Camera angle is extreme

### Similar Packaging

Products with similar packaging may occasionally be confused by the product recognition model.

### AI-Generated Missing Product Images

For testing planogram compliance, some shelf images were generated by removing products from the original shelf images.

During this process, minor visual changes introduced by image editing may slightly alter product appearance.

As a result:

- Missing products are usually detected correctly.
- Some products may occasionally receive a different SKU prediction.
- This may lead to additional misplaced-product warnings even when the product is physically present.

### Fixed Shelf Library

The current version supports:

- Shelf A
- Shelf B
- Shelf C
- Shelf D
- Shelf E

### Processing Time

The first analysis may take longer because machine learning models must be loaded into memory before inference begins.
